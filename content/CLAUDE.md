# Claude

These are instructions for working in this repository for coding agents.

## Overview

Markdown source files for blog posts and articles.

## Working in this repository

- You are in a monorepo in the `content` folder. The root of the monorepo is at `..`.
- Always run pnpm commands from the monorepo root.

### Commands to use

- `pnpm run format` - Format markdown with Prettier
- `pnpm run format:check` - Check formatting
